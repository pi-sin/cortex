# Skill: Competitive Watch

When analyzing competitors (monthly or triggered by news):

## 1. Competitor Update

- What did they ship recently?
- Pricing changes?
- User sentiment (Twitter, Reddit, G2, ProductHunt comments)
- Funding raised?
- New integrations or partnerships?

## 2. Threat Assessment

- Does this threaten our core differentiator (AI + widgets)?
- Does this validate our direction?
- Does this change roadmap priorities?

## 3. Response Options

For each significant move:

- **Ignore** — not relevant to our users
- **Monitor** — watch for user reaction
- **Accelerate** — speed up a planned feature
- **Differentiate** — double down on what makes us different

## Competitors to Track

Shift, Rambox, Wavebox, Ferdium, Notion, Arc Browser, Apple (unified hub)

## Rules

- Never copy features just because competitors ship them
- Only copy if it solves a validated user pain point
- Our moat is AI intelligence — every response should reinforce that
