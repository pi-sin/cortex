# Skill: Investor Prep

When preparing fundraising materials:

## Pitch Structure (12 slides max)

1. Problem — fragmented tools (28 min/day reconnecting context, 36 app switches/hour)
2. Solution — Cortex: AI-powered unified workspace
3. Why Now — AI (Claude, GPT-4) just became good enough; predecessors failed without it
4. Market Size — TAM/SAM/SOM for productivity tools
5. Traction — beta users, engagement, testimonials
6. Product — live demo of key flows
7. Business Model — Free/Pro/Team, LTV/CAC math
8. Competition — why Shift/Rambox/Wavebox stalled
9. Team — why this team executes
10. Ask — amount, use of funds, milestones

## Common Objections (prepare answers)

- "Isn't this just a glorified browser?"
- "What if Google/Microsoft builds this?"
- "How do you handle API dependencies?"
- "What's your moat?"
- "Why desktop instead of web?"
- "Enterprise security requirements?"

## Key Metrics

- DAU/MAU ratio (target >40%)
- Integrations per user (target >3.5)
- Session duration (target >45 min/day)
- AI adoption rate, NPS (target >40)
