# Skill: Go-To-Market Strategy

When planning launch or growth:

## 1. Target User (be precise)

- Job title, company size, daily tool count
- Biggest daily frustration
- Where they hang out online

## 2. Distribution Channels (rank by ROI)

For each: action, cost (time + money), expected reach, timeline to results.
Evaluate: Product Hunt, Hacker News, Twitter/X, Reddit (r/productivity, r/macapps),
YouTube demos, newsletters, productivity influencers, SetApp, AlternativeTo, SEO, paid ads.

## 3. Beta Program

- Size: 50-200 users
- Recruitment source
- Feedback mechanism (in-app? Slack community? Typeform?)
- Duration: 4-8 weeks
- Beta-to-paid conversion target

## 4. Launch Sequence (4-week plan)

- Pre-launch: build anticipation
- Launch day: coordinate all channels
- Post-launch: sustain momentum, respond to feedback

## 5. Risks

What could make the launch fail? Mitigation for each.
